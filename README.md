# Advent of Code 

> [!NOTE]
> This repository has custom commands that you can use with `just`.
> To use them you'll want to download `just`

## Commands

To bootstrap a problem for a new year and day, run the following command

```bash
just new {{YEAR}} {{DAY}}
```

To run the configured problem from `main/src/main.rs`, run the following command

```bash
just run
```
